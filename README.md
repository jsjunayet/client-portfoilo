### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
    ```
  

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open the site in your browser:
   ```
   http://localhost:3000
   ```
